# DNS Server Project

A custom DNS server implementation built from scratch using Node.js.

## Description

This project demonstrates how to build your own DNS server using UDP sockets.

## Features

- UDP socket-based DNS server
- Listens on port 53
- Handles incoming DNS queries

## How to Run

```bash
node index.js
```

## Testing

Use nslookup to test the DNS server:

```bash
nslookup google.com 127.0.0.1
```
